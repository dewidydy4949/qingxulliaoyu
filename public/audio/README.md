# 音频文件目录

请将音频文件放置在此目录中。

## 推荐的音频文件

请准备以下音频文件并放置在此目录：

1. **深度放松冥想** (`meditation-1.mp3`)
   - 5分钟的冥想引导音频
   - 建议：舒缓的引导语，背景音乐轻柔

2. **星空助眠故事** (`sleep-story-1.mp3`)
   - 10分钟的睡前故事音频
   - 建议：温柔的声音，适合入睡

3. **森林白噪音** (`forest-white-noise.mp3`)
   - 15分钟的自然白噪音
   - 建议：森林环境音，鸟叫声等

## 音频格式要求

- 格式：MP3（推荐）或 WAV
- 采样率：44.1kHz 或更高
- 比特率：128kbps 或更高
- 文件大小：建议每个文件不超过 10MB

## 音频资源推荐

如果您需要免费的音频资源，可以参考以下网站：

1. **免费音效库：**
   - Freesound.org - https://freesound.org/
   - Pixabay - https://pixabay.com/music/
   - Free Music Archive - https://freemusicarchive.org/

2. **冥想和白噪音：**
   - YouTube Audio Library（可下载）
   - Zapsplat（注册后可下载）

3. **文本转语音（TTS）：**
   - 如果您的音频是文本内容，可以使用TTS服务生成音频
   - 如：Azure Text-to-Speech、Google Cloud TTS 等

## 文件命名规范

建议使用以下命名规范：
- 冥想音频：`meditation-[编号].mp3`
- 故事音频：`story-[编号].mp3`
- 白噪音：`white-noise-[类型].mp3`

## 注意事项

1. 确保音频文件有适当的版权许可
2. 音频文件路径在代码中配置，详见 `src/pages/p-home/index.tsx`
3. 上传音频后需要重启开发服务器才能生效
