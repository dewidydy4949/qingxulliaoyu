# 🎹 如何快速获取免费的舒缓钢琴曲

## 📥 方法一：从免费音乐网站下载（推荐，5分钟搞定）

### 步骤1：访问免费音乐网站

推荐以下网站（全部免费、免版权）：

1. **Pixabay Music**（最推荐）
   - 网址：https://pixabay.com/zh/music/
   - 搜索关键词：`piano`、`relaxing piano`、`meditation piano`
   - 点击音乐 → 点击"免费下载" → 选择MP3格式

2. **Freesound**
   - 网址：https://freesound.org/
   - 搜索关键词：`piano`、`relaxing piano music`
   - 注册账号（免费）→ 下载MP3文件

3. **Free Music Archive**
   - 网址：https://freemusicarchive.org/
   - 搜索关键词：`piano`、`ambient piano`
   - 直接下载MP3文件

### 步骤2：下载3首钢琴曲

建议下载：
- 1首舒缓的冥想钢琴曲（约5分钟）
- 1首宁静的助眠钢琴曲（约10分钟）
- 1首放松的环境钢琴曲（约15分钟）

### 步骤3：将文件放入项目

1. 将下载的MP3文件重命名为：
   - `piano-meditation-1.mp3`（深度放松冥想）
   - `piano-sleep-1.mp3`（星空助眠故事）
   - `piano-forest-1.mp3`（森林白噪音）

2. 将文件放入项目目录：
   ```
   public/audio/
   ├── piano-meditation-1.mp3
   ├── piano-sleep-1.mp3
   └── piano-forest-1.mp3
   ```

### 步骤4：更新配置文件

我已经为您准备好了配置，只需要：
1. 将文件放入 `public/audio/` 目录
2. 刷新浏览器即可自动使用本地文件

## 📥 方法二：使用在线工具快速下载（最简单）

### 使用 YouTube 转 MP3（仅用于个人项目）

1. 在 YouTube 搜索：`relaxing piano music`、`meditation piano`
2. 复制视频链接
3. 使用在线转换工具（如 y2mate.com）转换为MP3
4. 下载后放入 `public/audio/` 目录

**注意**：请确保遵守版权规定，仅用于个人项目。

---

## 📥 方法三：使用我提供的快速下载脚本

我已经为您创建了一个配置文件，您只需要：

1. 打开 `src/config/audioConfig.ts`
2. 将音频URL改为本地路径（我已经准备好了配置）
3. 下载文件并放入 `public/audio/` 目录
