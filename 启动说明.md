# 🚀 项目启动说明

## 开发服务器已启动

开发服务器正在后台运行中。

## 📍 访问地址

根据 Vite 的默认配置，开发服务器通常运行在：

**本地访问：** http://localhost:5173

**网络访问：** http://您的IP地址:5173

## 🔍 如何确认服务器已启动

1. **查看终端输出**
   - 如果看到类似以下输出，说明服务器已启动：
   ```
   VITE v7.0.3  ready in xxx ms

   ➜  Local:   http://localhost:5173/
   ➜  Network: http://192.168.x.x:5173/
   ```

2. **浏览器访问**
   - 打开浏览器
   - 访问：http://localhost:5173
   - 如果能看到页面，说明服务器正常

## 📝 访问页面

启动后，您可以访问以下页面：

- **登录页：** http://localhost:5173/login
- **首页：** http://localhost:5173/home
- **情绪选择：** http://localhost:5173/emotion-select
- **疗愈结果：** http://localhost:5173/result
- **历史记录：** http://localhost:5173/history
- **我的收藏：** http://localhost:5173/collection
- **设置：** http://localhost:5173/settings
- **反馈建议：** http://localhost:5173/feedback
- **帮助中心：** http://localhost:5173/help

## 🎵 测试音频播放

1. 访问首页：http://localhost:5173/home
2. 找到"为您推荐"部分
3. 点击推荐卡片上的播放按钮（▶）
4. 应该能听到音频播放

## ⚠️ 如果无法访问

1. **检查端口占用**
   - 默认端口：5173
   - 如果被占用，Vite 会自动使用下一个可用端口

2. **查看完整终端输出**
   - 查看是否有错误信息
   - 确认服务器实际运行的端口

3. **检查防火墙**
   - 确保防火墙允许访问该端口

## 🛑 停止服务器

在运行 `npm run dev` 的终端中按：
- `Ctrl + C` （Windows/Linux）
- 或关闭终端窗口

## 📋 常用命令

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview

# 运行代码检查
npm run lint
```
