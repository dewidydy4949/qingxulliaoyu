# 📁 需要准备的音频文件

## 文件清单

请将以下音频文件放入 `public/audio/` 目录：

### 必需文件：

1. **meditation-1.mp3**
   - 用途：深度放松冥想
   - 时长：约5分钟
   - 建议内容：冥想引导语、舒缓音乐

2. **sleep-story-1.mp3**
   - 用途：星空助眠故事
   - 时长：约10分钟
   - 建议内容：睡前故事、温柔的声音

3. **forest-white-noise.mp3**
   - 用途：森林白噪音
   - 时长：约15分钟
   - 建议内容：森林环境音、自然声音

## 📍 文件位置

```
项目根目录/
└── public/
    └── audio/
        ├── meditation-1.mp3          ← 放入这里
        ├── sleep-story-1.mp3         ← 放入这里
        └── forest-white-noise.mp3    ← 放入这里
```

## 🔄 如果文件不存在

如果本地文件不存在，系统会**自动使用在线备用音频**进行测试。

## ✏️ 修改文件名

如果您想使用不同的文件名，请编辑 `src/config/audioConfig.ts`：

```typescript
{
  id: '1',
  audioUrl: '/audio/您的文件名.mp3',  // 修改这里
}
```

## 📝 文件格式要求

- **格式**：MP3（推荐）或 WAV
- **采样率**：44.1kHz 或更高
- **比特率**：128kbps 或更高
- **文件大小**：建议每个文件不超过 10MB

---

**提示**：即使没有这些文件，音频播放功能也可以正常工作（会使用在线备用音频）。
