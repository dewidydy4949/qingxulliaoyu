# 部署到 Vercel 获取公开网址

## 快速部署步骤：

### 1. 访问 Vercel
打开浏览器，访问：https://vercel.com

### 2. 登录账号
- 点击 "Sign Up" 或 "Log In"
- 选择使用 **GitHub** 账号登录（推荐，因为代码已经在 GitHub 上）

### 3. 导入项目
- 登录后，点击 "Add New..." → "Project"
- 在 "Import Git Repository" 中，找到你的仓库：`dewidydy4949/app_623430599938`
- 点击 "Import"

### 4. 配置项目（通常会自动检测）
- **Framework Preset**: Vite（会自动检测）
- **Root Directory**: `./`（默认）
- **Build Command**: `npm run build`（默认）
- **Output Directory**: `dist`（默认）
- 其他设置保持默认即可

### 5. 部署
- 点击 "Deploy" 按钮
- 等待 1-2 分钟，部署完成后会显示你的公开网址
- 网址格式类似：`https://app-623430599938.vercel.app`

### 6. 访问你的应用
部署完成后，你会得到一个公开网址，任何人都可以通过这个网址访问你的应用！

---

## 方法 2：使用 Vercel CLI（需要登录）

如果你想使用命令行部署：

1. 在终端运行：`vercel login`
2. 在浏览器中完成登录
3. 运行：`vercel --prod`
